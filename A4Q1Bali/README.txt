Jaivir Bali (7775370) A4Q1

TEXTURES:
All textures obtained from textures.com

BINDINGS:
'w' - forward 1 unit
's' - backward 1 unit
'a' - left 1 unit
'd' - right 1 unit
' ' - switch perspectives

GAME:
Start at forward position. Can only move back and then forward from that back unit.
Start at middle. Can move between left, middle and right tiles
The ship will move by default a tile every 0.5 seconds.
Collisions pause the movement of the game, move away (side/back) for game to continue.
Start with axonomic perspective, hitting space bar gets you to 3rd person overhead.


EXTRA FEATURE:
Added particle system to ship so it looks like it's exhausting flames
